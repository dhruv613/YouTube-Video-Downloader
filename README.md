# 📘 Overview

YT Video Downloader is a lightweight Python-based tool that allows users to:

- Download YouTube videos
- Select custom video quality
- Save videos automatically to a downloads folder
- Use best-quality merging with FFmpeg
- Download content quickly using yt-dlp

The project is designed to be:

- Beginner friendly
- Fast
- Flexible
- Extendable for future desktop applications

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/dhruv613/YT-Video-Downloader.git
cd YT-Video-Downloader
```

---

## 2️⃣ Install Dependencies

```bash
uv pip install -r requirement.txt
```

---

# ▶️ Usage

Edit the configuration file:

```python
VIDEO_URL     = "your_youtube_url_here"
OUTPUT_FOLDER = "downloads"
QUALITY       = "best"
```

Run the downloader:

```bash
python Downloads.py
```

---

# ⚙️ Configuration

| Variable | Description |
|----------|-------------|
| VIDEO_URL | YouTube video URL |
| OUTPUT_FOLDER | Download folder |
| QUALITY | best / worst / format code |
